the keyboard keys qwertyuiop[] and asdfghjkl;' and zxcvbnm,./ will let you activate the first through third chord rows. the strumplate on the right will let you play notes within four octaves of those chords. if you press the first and second row keyboard keys at the same time, the fourth row will activate. the fourth row is connected to the first keyboard on the right side, and you can put in different chords there. the fifth row is likewise connected to the second keyboard on the right side.

the keyboard at the bottom lets you input a sequence in and it will also be reflected on the orange strumplate to the left of the main one on the right.  if you press the 1 key, it will insert a 16th note rest denoted by an x in the text box above it. if you press the 2 key (on the pc keyboard), it will delete the previous note. also, 3,4,5,and 6 will activate the 8th, 16th, dot 8th, and trip 8th buttons. 1 will always insert a 16th note rest unless the trip 8th button is active in which case it will insert a x# which is a triplet 8th rest. the aqua colored number to the right of the sequence input box shows the length of the sequence, so it should probably be something like 32 or 64 before you end up hitting the SEQW button, which adds a colored box that has that sequence in it. once you have more than one sequence box, MOVL and MOVR can let you move them around, and SEQD will delete the highlighted box (click on them to highlight them). SEQC will clone the highlighted sequence box, and SEQR will read the sequence into the sequence text box. PLAY SONG plays all of the sequence boxes in order, hwereas PLAY SEQ plays the sequence that's inside of the box above the large keyboard. you can use SEQR and SEQU to update a sequence, SEQU updates the highlighted sequence to the one that's inside of the text box.

the REC button records what's happening, you can also record a sequence playback or song playback. PLAY will playback that recording, and STOP will stop the playback. RENDER will go ahead and playback the Recording and also when you hit RENDER again, it will end the recording and let you save a .wav file of it.  you can also play the strumpad and stuff during the playback of the recording and it will show up on the render I think, as well as play after the recording has played. also you can export the MIDI file and load a MIDI file as well, although I haven't loaded any MIDI files other than the ones that I've made in this omnichord program.  the TAP BPM button lets you tap the BPM and the metronome flashes but doesn't make noise. you'll see that the 1st beat of 4 looks different. this could be important if you want to use the MIDI export to synchronize it.

otherwise you should also know that the page asks you to access your MIDI devices because it's able to Send MIDI to external devices such as a synthesizer.  I'm using LoopMIDI to use this with Renoise to record chord strums.

also there is an XYPAD button. if you hit this there will be a sort of XYPAD covering the chord section, and you can select different scales using that dropdown below it. I would go with Harmonic Minor or Ionian Major. and the X and Y sliders next to the XYPAD button will control the number of vertical or horizontal sections in the XYPAD. also you can toggle SEL to select a particular square in the XYPAD and use SUP to increase its semitone value by 1 or SDN to decrease its semitone value by 1. the small text in the squares denotes the midi note value and 60 happens to be middle C. however the Octave and Transpose buttons at the top of the control section will let you change the octave and transpose by semitones. also the Audio Sus slider controls the sustain of the sound that the .htm page makes, and the MIDI OFF slider controls the length of the actual MIDI note sustain.

if you want to disable the second strumplate, delete the text inside of the text input for the sequencer portion. also the entire song and state of the file can be saved by using the Copy button which will copy the stuff in the text box next to it to the clipboard, and then you can save it to a text file, and then when you reopen this omnichord htm file you can then paste it into the text box next to the Load button and hit Load and it will load it!
